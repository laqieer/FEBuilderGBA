#1377 Class Editor Battle Anime Jump -- before/after proof (real FE8U ROM, Avalonia headless)

The Class Editor's BattleAnime (P52) Jump passes the class's battle-anime SETTING pointer.
OLD: NavigateTo -> EntryList.SelectAddress(settingPtr) finds no row -> stays pinned on entry 0
     (0x00C00008, "0x00 Anim") -> "No animation data found for this ID (ID=0...)".
NEW: NavigateTo resolves the owning class (ClassFormCore.GetIDWhereBattleAnimeAddr) and
     DIRECT-LOADS the setting pointer -> the editor shows that class's real animation.
