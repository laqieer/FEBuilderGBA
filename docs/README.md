# Documentation index

This directory holds the **authoritative, living documentation** for
FEBuilderGBA. Start at the top-level [`README.md`](../README.md) for the project
overview and quick start; the documents below go into depth on specific areas.

> Completed, point-in-time process docs (Phase-0 migration, split-package work,
> one-off status/verification reports) have been moved to
> [`archive/`](archive/README.md) for provenance.
