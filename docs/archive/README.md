# Archived documentation (historical)

This directory holds **point-in-time process/working documents** that described
migrations and refactors that have since **completed**. They are kept for
**provenance only** — they are no longer maintained and are **not** part of the
authoritative living documentation set (see [`../README.md`](../README.md) for
the docs index).

> **Note on internal links:** these snapshots were written when the files lived
> directly under `docs/`. Some of them reference sibling files using the original
> `docs/…` paths (e.g. `` `docs/phase0-migration-guide.md` ``). After archiving,
> those references resolve relative to the *original* layout, not this `archive/`
> folder, so do not rely on them as working links — treat the text as a
> historical record. The files they mention now live alongside this README in
> `docs/archive/`.
